
通过用户名/密码进行一种基本的安全配置。


|**参数**|**默认值**|**描述**|
|:-|:-|-
| type               | **basic**                      | Type of authorization.      |
| username           | **username**                   | Username for authorization. |
| password           | **password**                   | Password for authorization. |
|---

security示例如下：

```yaml
    "security": {
      "type": "basic",
      "username": "username",
      "password": "password"
    }
```
